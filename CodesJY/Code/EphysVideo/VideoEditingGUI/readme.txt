建议先将seq文件及相应的idx文件复制至C/D盘后开始运行,可以提高3-4倍转为jpg的速度
1. 新建项目/加载项目。区别在于是否使用设定好的config.mat
2. 将seq文件转为jpg文件，仅支持每秒100帧的视频，初始参数为读至180000帧。对于一些seq文件并不能按要求读完全部的seq文件，可手动减小end_reading参数或删除损坏的文件。（需要约6min）
3. 通过“寻找灯位置”按钮按照提示寻找出现灯的图片中的灯位置。选定阈值选取方式。手动选取需输入特定的阈值，自动选取可自动选取阈值（大概率成功），随后选取可按照之后产生的figure程序中断输入阈值（成功率最高）
4. 帧率为每秒的帧数，默认为100（原速）。向前(s)与向后(s)为所需截取的avi文件的长度。以灯亮的一瞬间为0点，截取向前a秒至向后b秒的视频。（约8min）
5. 选择对应的med与bpod文件，完成视频分类（若出现多余/缺少的avi视频将出现错位现象）
